Documentation Plugin SONOS pour le projet S.A.R.A.H


Auteur : Sylvain Chauffaille Basé sur le travail de Sébastien ANESE
date : 16/02/2014
version : 1.0


Ce plugin permet de contrôler des périphériques SONOS. Il vous faut dans un premier temps récupèrer l'IP de vos équipements SONOS sur le réseau.

Les actions suivantes sont disponibles :
- lecture (play)
- pause (pause)
- musique suivante (next)
- musique précédente (previous)
- augmenter le volume
- baisser le volume


Dans l'application SONOS sur un PC, allez dans "Aide" puis "A propos de mon système SONOS". Vous y verrez ici l'ensemble des périphériques.


Récupèrer les IP et renseignez les dans le fichier "sonos.prop" présent dans sarah/plugins/sonos.

Attention a bien faire correspondre le lieu du fichier "sonos.xml" (variable : out.action.idPiece='salon') avec la variable présente dans le fichier de configuration (ip)

Vous pouvez commander vos sonos en indiquant à chaque fois l'action et le lieu. Sarah vous répondra dans le lieu en question
Exemple :
Sarah --- Action --- lieu
Sarah --- Allume la musique --- dans le salon
